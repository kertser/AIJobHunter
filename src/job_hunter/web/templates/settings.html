ℹ️ How to export LinkedIn cookies

Option A — Browser extension
Install a cookie export extension (e.g. Cookie-Editor, EditThisCookie), go to linkedin.com while logged in, click the extension and export all cookies as JSON.

Option B — DevTools
Open DevTools → Application → Cookies → https://www.linkedin.com. Copy all cookies into a JSON array of {"name", "value", "domain", "path"} objects.

🖥️ Remote Login (log in directly from the server)

Log in to LinkedIn using a headless browser on the server. Use this when you can't export cookies from a local browser (e.g. Docker deployment).

⚠️ What to expect

LinkedIn may ask for a verification code sent to your email or phone. The system will automatically click "Send code" if possible. Watch the log below for status.

If you don't receive a code, LinkedIn may be blocking automated login from this IP. In that case, use the cookie upload method above instead.

If LinkedIn shows a CAPTCHA or "approve from app" prompt, programmatic login won't work — use cookie upload.
